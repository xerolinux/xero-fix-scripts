# Collection of some scripted fixes

**Terminal Commands :**

- ram : Uses _rate-mirrors_ to update **Arch** mirrorlist.
- smbfix : Fixes any issues with **Samba** Windows shares.
- togdm : Switches your _Display/Login Manager_ to **Gnome**'s **GDM**.
- tolightdm : Switches your _Display/Login Manager_ to **Mint**'s **LightDM**.
- tosddm : Switches your _Display/Login Manager_ to **KDE Plasma**'s **SDDM**.
